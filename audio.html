Lossy and Lossless Audio

Lossless Audio Files

Examples of lossless file types

  • WAV
  • FLAC
  • AIFF

Why use Lossless file formats?

You will generally use Lossless file types because when it is compressed all the original data is there. Lossless files are mainly used for text or spreadsheet files or files that you wish to compress and not lose anything.

Lossy Audio Files

Examples of Lossy file types

  • MP4
  • MP3
  • OGG

How does Lossy compression work?

Lossy compression reduces the file size by eliminating certain information that is not needed and won't be noticed when eliminated.

Why use Lossy file formats?

We use Lossy formats because when we compress it it can be made into smaller files and then you can have more files for music,videos and more.
